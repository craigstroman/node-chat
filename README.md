# Node Chat

A chat room using Node.js and Socket.io.  Based on the tutorial from https://www.sitepoint.com/chat-application-using-socket-io/.

### To run locally:
- Open terminal and run git clone https://github.com/craigstroman/node-chat.git.
- Then cd node-chat and run npm install and also bower install.
- To start it then run npm start in the terminal.
